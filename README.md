Pour le lancement de l'API : Pour que l'api du site soit active, dans le terminal éxécuter cette commande :
"php -S localhost:8000"

Si le message afficher après exécution de la commande, ressemble a celui-ci :
"[Tue Mar  3 13:58:20 2026] PHP 8.4.11 Development Server (http://localhost:8000) started"

Alors le serveur local est bien actif.
